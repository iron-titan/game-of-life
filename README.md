## How to Run Locally

This app includes a dedicated local server script to run safely on your machine without external hosting.

1. **Ensure you have Python installed** (standard on macOS).
2. **Run the start script:**
   ```bash
   python3 start_game.py
   ```
   This will automatically launch the app in your default web browser.

3. **To Stop:** Press `Ctrl+C` in the terminal window.
